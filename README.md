# frontend-craft

A Claude Code skill for building distinctive, production-grade frontend interfaces with deep craft methodology.

Combines expressive design direction, intent-driven methodology, systematic self-evaluation, and practical tooling to produce interfaces that avoid generic AI aesthetics.

## Scope

Handles two modes:

- **Interface Design** — Dashboards, admin panels, SaaS apps, tools, settings pages, data interfaces. Leans on craft foundations: intent, consistency, memory, systematic tokens.
- **Expressive Design** — Landing pages, marketing sites, posters, editorial layouts, one-off web experiences. Leans on bold aesthetic direction: dramatic typography, unexpected layouts, atmospheric effects.

Both modes share domain exploration, anti-default methodology, and self-evaluation checks.

## Installation

Clone into your Claude Code skills directory:

```bash
git clone git@github.com:samuelralak/frontend-craft-skill.git ~/.claude/skills/frontend-craft
```

Claude Code will pick it up automatically on the next session.
